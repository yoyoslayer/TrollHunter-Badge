# TrollHunter-Badge
- Slack Username: @Yousif

# What it is
It is a PCB that holds an NFC chip that links to the TrollHunter Netflix site so people can start watching the show Immediately. Furthermore, once they click on the show, netflix will be more encouraged to show them the show again so they are more likely to actually watch it. 

# Why I made it
This was the opening challenge for Day 1 of the IRL weeklong hackathon held by Fallout. A program created from the HackClub organization where attendees record 60 hours of electronic work and get a fully paid 1-week hackathon stay in Shenzhen, China. I chose TrollHunters because I want to spread the enjoyment of the show to everyone I meet. It is one of my favorite shows growing up and I wish the fandom/community was bigger.

# How to create yourself
1. Download the Project files and Gerber
2. Import the files into EasyEDA
3. Double Check the LCSC part for the NFC Chip
2.5. Add any additional silkscreen art (Optional)
4. Export the project to JLCPCB
5. Order the PCB with assembly
